js处理上灵活性比较大，本网站采用的是Jquery，因为它有丰富的插件选择。

由于ajax操作比较频繁，所以我封装了ajax方法到jquery类方法中，如果你了解js的话，将对js对象继承不陌生。

jquery source
$(function () {
  $.extend({
    myajax: function (data) {
    $.ajax({
    type: 'get',
    url: data,
    dataType: 'text',
    success: function (data) {
    $("#content").html(data);
    }
    });
  }
  });
})
